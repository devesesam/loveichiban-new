# Love Ichiban Website

Static website for **Ichiban Poke & Donburi** — Authentic Japanese Street Food in Tauranga, New Zealand.

## 🍜 Live Site

[loveichiban.co.nz](https://loveichiban.co.nz/)

## Tech Stack

- Pure HTML, CSS, JavaScript
- No build tools or frameworks required
- Hosted on [Netlify](https://netlify.com)

## Pages

- **Home** — Hero, intro, services, gallery, map
- **Menu** — Donburi, Poke Bowls, Sauces, Rice Options
- **About Us** — Team story

## Local Development

```bash
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000)

## Deployment

This site is configured for Netlify with clean URLs via `netlify.toml`. Simply connect this repo to a Netlify project and it will auto-deploy.
